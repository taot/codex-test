# OpenCode for PyCharm (Cursor-like UI)

A JetBrains/PyCharm plugin prototype that recreates key Cursor-style GUI workflows:

- Sidebar chat with conversation list
- Context panel for selected files/instructions
- Inline prompt action for selected code
- Command palette action to open the assistant

## Run

```bash
./gradlew runIde
```

## Build

```bash
./gradlew buildPlugin
```
